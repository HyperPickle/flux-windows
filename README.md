# Flux for Windows

Flux has been ported from macOS (Swift) to Windows (Python + PyQt6).

## Requirements

- Python 3.9+
- pip (Python package installer)

## Installation

Install dependencies:
```
pip install -r requirements.txt
```

## Running the App

```
python src/main.py
```

The app will appear in your system tray. Click the tray icon to open the Flux window. Right-click the tray icon to quit.
